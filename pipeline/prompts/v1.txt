## Provision Keys (extract all that appear in the document)

## Rules

1. If a field is absent or cannot be determined from the text, set it to null.
2. clause_excerpt MUST be verbatim text from the contract, truncated to ≤ 80 words. Never paraphrase.
3. confidence reflects how certain you are the extracted value is correct: 0.95+ = explicit numeric statement; 0.80–0.94 = clearly implied; 0.50–0.79 = inferred or ambiguous; < 0.50 = very uncertain (do not omit — include with low confidence).
4. If the PDF contains more than one collective bargaining agreement (e.g., teaching staff AND classified staff), output one contract object per agreement in the "contracts" array.
5. Do not fabricate values. Do not infer percentages from incomplete data.
6. Salary schedule amounts should be the BASE values (first step, BA column) unless the key specifies max.
